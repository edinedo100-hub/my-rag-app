# Competitor Analysis: WattWise vs. Energy Management Market

## Market Context
The energy management app market is shifting from simple bill tracking to **proactive automation** and **AI-driven savings**. While WattWise has a strong foundation in human-verified referrals and bill uploads, several tech-heavy competitors (e.g., Rocket Money, Energy Ogre, Sense) are setting new benchmarks for automation.

---

## 1. Feature Gaps (What competitors have, we don't)
*Sorted by Importance/Urgency*

| Feature | Importance | Description |
| :--- | :---: | :--- |
| **Instant AI OCR Extraction** | **CRITICAL** | Automated data extraction from uploaded bills (usage kWh, total cost, tariff name). Eliminates the "Pending" wait time for manual admin review. |
| **Usage Analytics & Forecasting** | **HIGH** | Interactive graphs showing historical usage patterns and AI-based cost predictions for the upcoming month. |
| **Direct Rate Comparison** | **HIGH** | A built-in "Switching Engine" that compares the user's current extracted rate against live market offers. |
| **Smart Meter Integration** | **MEDIUM** | API connection to national smart meter grids or hardware (like Sense) for real-time (minute-by-minute) usage monitoring. |
| **Account Sync (Plaid/Banking)** | **MEDIUM** | Automatically detecting utility payments in the user's bank feed to update the dashboard without manual uploads. |
| **Automated Negotiation** | **LOW** | Services that contact the provider on the user's behalf to negotiate "loyal customer" discounts. |

---

## 2. Competitive Edge (Features NO ONE has yet)
*Sorted by Strategic Impact*

| Innovative Feature | Strategic Value | Description |
| :--- | :---: | :--- |
| **"Neighborhood Power" Bulk Deals** | **GAME CHANGER** | Aggregating users in a specific ZIP code to negotiate custom, volume-based tariffs with providers. "Retail power at wholesale prices." |
| **AR "Energy Ghost" Scanner** | **HIGH** | Use the phone's camera (AR) to point at appliances. The app overlays the real-time cost ($/hr) to run that specific device based on your current tariff. |
| **Green Multiplier Referrals** | **MEDIUM** | Dynamic referral rewards: Earn 1.5x balance if your referral switches to a 100% renewable energy plan. Brand positioning as "The Green Switcher." |
| **AI Energy Agent (Voice/Proactive)** | **MEDIUM** | A proactive AI assistant that detects anomalies (e.g., "Usage is spiking - did you leave the stove on?") and provides actionable advice via voice. |
| **Cross-Provider "Credit Bridge"** | **LOW** | Allow users to use referral credits earned on one utility (e.g., Electricity) to pay for another (e.g., Water or Gas) via a unified wallet. |

---

## Recommendations for WattWise Roadmap
1.  **Immediate Priority**: Implement **AI OCR**. Manual admin review for bill uploads will become a bottleneck as the app scales. This will make the user "Bill Upload" experience feel instantaneous.
2.  **Growth Priority**: Implement the **"Green Multiplier"**. It aligns with ESG (Environmental, Social, and Governance) trends and makes the referral system more viral and "mission-driven."
3.  **Long-term Advantage**: Explore **"Neighborhood Power"**. This creates a network effect—users will actively recruit their actual neighbors to lower their own bills, creating hyper-local density.

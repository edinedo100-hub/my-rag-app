# Project Roadmap

This document outlines the high-level phases of the WattWise development journey.

## Phase 1: Core Navigation & MVP Foundations (In Progress)
- [x] Refine upload navigation flow (allow routing to dedicated upload section without breaking back-navigation capabilities).
- [x] **Referral system — €10 credit:** Referrer's wallet credited when someone registers with their code or link (`?ref=CODE`). Works for email signup AND Google Sign-In.
- [ ] Stabilize app functionality, ensuring robust manual referral and simple bill tracking works flawlessly.

## Phase 2: Instant AI & Automation Layer (Next Up)
- [ ] **Instant AI OCR Extraction (CRITICAL):** Automate data extraction (usage kWh, total cost, tariff name) from uploaded bills.
- [ ] **Usage Analytics & Forecasting:** Add interactive graphical tracking for historical usage patterns and cost predictions.
- [ ] **Direct Rate Comparison:** Build a "Switching Engine" linking the user's current rate against live market offers.

## Phase 3: Hardware & Deeper Integrations
- [ ] **Smart Meter/Plaid Integration:** API connections for real-time monitoring and bank-feed detection of utility payments to automatically update dashboards.
- [ ] **Green Multiplier Referrals:** Dynamic referral rewards prioritizing renewable energy connections.

## Phase 4: Long-Term Differentiators
- [ ] **Neighborhood Power Bulk Deals:** Aggregating local ZIP codes for custom wholesale tariffs.
- [ ] **Energy Ghost (AR Scanner):** Overlay device power usage ($/hr) through AR camera.
- [ ] **AI Energy Agent:** Proactive voice-based anomaly detection (e.g., usage spikes).

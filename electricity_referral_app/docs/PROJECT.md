# Project Vision: WattWise (Electricity Referral App)

## Overview
WattWise is an energy management and electricity referral application designed to shift the market from simple bill tracking to proactive automation and AI-driven savings. The core value proposition currently revolves around human-verified referrals and bill uploads, but the goal is to transform it into an automated, highly intelligent energy savings tool that leverages features like AI OCR and neighborhood power bulk deals.

## Goals & Objectives
1. **Reduce Friction:** Eliminate wait times for user bill uploads by moving from manual backend reviews to Instant AI OCR extraction.
2. **Drive Viral Growth:** Leverage innovative models like "Green Multiplier Referrals" and "Neighborhood Power Bulk Deals" to establish network effects and user acquisition at scale.
3. **Data-Driven Savings:** Provide users with usage analytics, historical patterns, and rate savings through comparisons with current market offers.

## Core Directives
- **Spec-Driven Development:** We manage AI coding sessions by rigorously breaking work into atomic, verifiable phases.
- **Continuous State Maintenance:** Avoid "context rot." Information that outlasts a single agent session must be documented and updated in these GSD markdown files.
- **Maintain High Quality UI/UX:** Ensure fluid user journeys across all application modules (e.g., maintaining proper navigation when working within the "Upload Bill" flow).

1 Create a GitHub Token

  1. Go to: github.com/settings/tokens?type=beta
  2. Click "Generate new token" (Fine-grained)
  3. Name it: Gemini CLI Workshop
  4. Expiration: 90 days
  5. Repositories: All repositories
  6. Set permissions:
    • Contents: Read and write
    • Issues: Read and write
    • Pull requests: Read and write
    • Metadata: Read-only
  7. Click Generate token
  8. Copy the token (starts with github_pat_...)

2 Update Your Configuration

Option A: Copilot Endpoint

"github": {
  "httpUrl":
    "https://api.githubcopilot.com/mcp/",
  "headers": {
    "Authorization":
      "Bearer github_pat_YOUR_TOKEN"
  },
  "timeout": 5000
}

Requires GitHub Copilot subscription

Option B: Open-Source Server

"github": {
  "command": "npx",
  "args": [
    "-y",
    "@modelcontextprotocol/server-github"
  ],
  "env": {
    "GITHUB_PERSONAL_ACCESS_TOKEN":
      "github_pat_YOUR_TOKEN"
  }
}

Requires Node.js. No Copilot subscription needed.

3 Verify It Works

Start Gemini CLI and try these prompts:

> List my GitHub repositories
> Show recent issues in my workshop repo
> What's the latest commit on the AI workshop repo?

If it works, you'll see your repositories and issues listed directly in the terminal.

If it doesn't work → check the troubleshooting slide at the end

Complete settings.json

  "selectedAuthType": "oauth-personal",
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "github_pat_YOUR_TOKEN"

For Students: Quick Setup

If students want to use GitHub MCP on their own machine:

1 Install Gemini CLI (requires Node.js)

2 Create a GitHub token (Settings → Developer settings → Fine-grained tokens)

3 Add to their ~/.gemini/settings.json:

{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "THEIR_TOKEN"
      }
    }
  }
}

4 Restart Gemini CLI → try "List my repositories"

Troubleshooting

Problem Solution
"MCP server not found" Make sure Node.js is installed: node --version
"Authentication failed" Token may have expired — generate a new one
"Timeout" Increase timeout: "timeout": 10000
"Permission denied" Check token permissions (Contents, Issues, PRs)
Server won't start Test manually: npx -y @modelcontextprotocol/server-github

If the server starts manually without errors, the issue is in your token or config path.

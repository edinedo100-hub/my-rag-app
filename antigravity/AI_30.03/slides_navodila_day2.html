How %%writefile Works

In Jupyter:

A cell with %%writefile app.py at the top does not run the code — it writes it to a file instead.

Each step adds new code to app.py, which you then run with Streamlit.

%%writefile app.py
import streamlit as st

st.title("AI Explorer")
st.write("Hello from FAMNIT!")

Then in the next cell:

!streamlit run app.py &

Tips for Working

Read the errors!

Python tracebacks are your friends. Read the last line — that's where the answer is.

Run cells in order

Don't skip cells. Each cell builds on the previous one. Skipping will break the code.

Experiment!

Change the parameters! What happens if you increase the number of neurons? What if you reduce the overlap?

Ask for help

Use Antigravity AI for coding help. Or raise your hand — we're here for you!

Save regularly

Colab can disconnect. Ctrl+S saves regularly! You can also use File → Save.
